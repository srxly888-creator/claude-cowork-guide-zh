# Claude Cowork 速查表

> **打印备用**

---

## ⚡ 快速启动
| 步骤 | 操作 |
|------|------|
| 启用 Cowork | 设置 → 功能 → 启用 Cowork |
| 授权文件夹 | 选择专用文件夹（不要用 Documents） |
| 第一个任务 | `"整理这个文件夹" |
| 审查计划 | 駻确认后按继续 |

---

## 📁 文件操作
### 埥看文件
```text
列出 [文件夹路径] 中的所有文件
```

### 读取文件
```text
读取 [文件路径] 的内容
```

### 创建文件
```text
创建 [文件名].[扩展名]， 内容： [内容]
```

### 编辑文件
```text
在 [文件路径] 中找到 [文本] 并替换为 [新文本]
```

### 重命名文件
```text
将 [旧文件名] 重命名为 [新文件名]
```

### 移动文件
```text
 将 [源文件] 移动到 [目标文件夹]
```

### 删除文件
```text
 删除 [文件路径]
⚠ 注意： 删除无法撤销
```

## ⚠ 安全须知
### 不要授权
- 整个 Documents 文件夹
- 桌面
- 系统文件夹
- 包含密码的文件夹

### 不要存储
- 密码
- API Key
- 银行信息
- 个人身份信息
### 始终备份
删除文件前先备份！Cowork 没有撤销功能
```

---

## 🔧 故障排除
| 问题 | 解决方案 |
|------|---------|
| VPN 不工作 | 断开 VPN |
| 使用量超限 | 等待 5 小时重置 |
| 文件访问失败 | 检查权限设置 |
| 输出格式错误 | 明确指定格式 |
| 插件不加载 | 重新安装插件 |
